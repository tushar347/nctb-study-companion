# NCTB Study Companion — Week 1 Starter

A course-project prototype for **Class 6 English for Today Line-by-line Study Companion**.

## What works in this starter

- Home screen with demo student profile
- Book library with Class 6 English for Today active
- Lesson reader with selectable line blocks
- Pop-up actions: Explain, Translate, Grammar, Quiz
- Side Help Tab with prepared helper output
- Short quiz with local score calculation
- Progress report with mock daily/weekly data

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Week 1 rule

Keep everything local and mock-data based. Backend, database, and AI API are Week 2+ tasks.
