# Week 3 Update — Book PDF Viewer

## Project Name

NCTB Study Companion  
Class 6 English for Today Line-by-line Study Companion

## Week 3 Goal

The goal of Week 3 is to add book PDF support to the prototype. Until Week 2, the app used structured Lesson 1 text blocks. In Week 3, we are preparing the app to support the original Class 6 English for Today textbook PDF while keeping the selected-line helper flow for the demo.

## Why This Is Needed

The project is a textbook companion, so the app should show a connection with the actual textbook. Adding PDF support makes the demo more realistic. Students can open the book PDF and also use the structured Lesson 1 reader for line-by-line help.

## Current Decision

For public GitHub safety, the full textbook PDF will not be committed immediately. The project will use a prepared `/public/books` folder and an official NCTB source note. If required for classroom demo, the PDF can be placed locally as:

`public/books/class6-english-for-today.pdf`

At the end of Week 3, the app should include:

- A clear separation between PDF reading and line-by-line helper mode.
