# Week 4 Update — Research-Oriented Agentic AI and RAG Learning System

## Project Name

NCTB AI Study Companion

## Current Branch

week4-research-features

## Main Goal of Week 4

The main goal of Week 4 is to convert the project from a normal OCR-backed textbook reader into a research-oriented AI learning system.

In Week 1–3, the project already included:

- Frontend prototype
- Book library and reader interface
- Backend API routes
- OCR-backed textbook content
- Lesson-wise reading system
- AI helper direction
- Dynamic quiz and report planning

From Week 4, the project is being upgraded toward a publication-standard research prototype. The focus is now on agentic AI workflow, RAG-based textbook grounding, student memory, research event logging, dynamic grammar learning, adaptive quiz generation, and learning analytics.
